# Defi-Watcher

Defi-Watcher is a **real-time monitoring tool for decentralized finance (DeFi) protocols**.  
It allows you to track on-chain events, liquidity movements, and other important activities in the DeFi ecosystem.  
This project is built using **Node.js** and **TypeScript**, making it fast, reliable, and easy to extend.

---

## Features

- Monitor real-time transactions and events on DeFi protocols  
- Track wallet activities and liquidity changes  
- Configurable alerts for specific events  
- Lightweight and modular design for easy customization  
- Written in TypeScript for type safety and maintainability

---

## Installation

1. **Clone the repository:**

```bash
git clone https://github.com/FrankezeCode/Defi-watcher-.git
cd Defi-watcher
```



2. **Install dependencies**

```bash
npm install
```

## Usage

- Customize the `src/config.ts` file to configure which DeFi protocols or wallets to monitor.
- Set alerts, logging preferences, and thresholds for events.
- The project is modular, so you can add new event handlers or integrate with other systems easily.
